# Professional Portfolio

Personal portfolio site for Euan Hill — AI Consultant and Senior Consulting Manager at Roboyo.

## Why I built this

After eight years leading AI and automation programmes across financial services, pharma, manufacturing, and telecom, I wanted a space that shows what I've delivered rather than just listing job titles. This portfolio is designed to give potential clients and collaborators a clear picture of the work I do: enterprise AI strategy, multi-agent platform design, and hands-on delivery that moves the needle.

I also wanted to build it myself. I spend my days designing AI solutions for others — this was a chance to ship something personal, experiment with modern frontend tools, and keep my hands on the code.

## What's in it

- **Career timeline** covering roles from management consulting through to senior consulting management
- **Selected engagements** including multi-agent AI platforms, large-scale process studies, and greenfield product builds
- **Personal projects** like CatFinder (AI-powered cat adoption matching) and Bergen (a travel companion app)
- **Smooth scroll animations** powered by GSAP and Lenis for a polished browsing experience

## Tech stack

- [Next.js 16](https://nextjs.org/) with the App Router
- [React 19](https://react.dev/)
- [TypeScript](https://www.typescriptlang.org/)
- [Tailwind CSS 4](https://tailwindcss.com/)
- [GSAP](https://gsap.com/) for scroll-driven animations
- [Lenis](https://lenis.darkroom.engineering/) for smooth scrolling

## Getting started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view it locally.

## Deployment

This site is deployed on [Vercel](https://vercel.com). Push to `main` to trigger a production deployment.
